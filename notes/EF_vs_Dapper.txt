EF vs Dapper?

EF требует высокие версии .Net для Sqlite, но чуть удобнее для использования.

Dapper менее удобен, но работает с .Net Framework 4.0.

Решение. Для публичных приложений использовать Dapper, чтобы не заставлять пользователей ставить лишний софт. Для своих внутренних утилит использовать EF.
